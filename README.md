# FusaoEmpresas

- Este projeto foi desenvolvido como estudo de manipulação e integração de dados em Python.
- O desafio consiste na fusão de dados de duas empresas que utilizam formatos diferentes de armazenamento:
- Uma empresa com dados em CSV e outra empresa com dados em JSON
- O objetivo foi ler, tratar e unificar essas informações em um único conjunto de dados, simulando um cenário real de fusão empresarial.

# Contexto do Desafio

## Em processos de fusão, é comum que empresas utilizem sistemas e formatos de dados diferentes. Neste projeto, o principal desafio foi:
- Ler arquivos em formatos distintos (CSV e JSON);
- Padronizar os dados para que tivessem a mesma estrutura;
- Resolver inconsistências entre os campos;
- Unificar as informações em um único modelo.
- O foco foi praticar lógica de programação, manipulação de estruturas de dados e organização de código.

# Tecnologias Utilizadas
- Python
- Manipulação de arquivos CSV e JSON utilizando bibliotecas nativas do Python (csv e json)
- Execução em ambiente Linux via WSL2

# Este projeto foi criado com foco em:

- Praticar leitura e escrita de arquivos
- Trabalhar com listas e dicionários em Python
- Resolver problemas de padronização de dados
- Simular um cenário real de integração de sistemas
